\doxysubsubsection{\texorpdfstring{timeout\_seconds}{timeout\_seconds}}
{\footnotesize\ttfamily double Auto\+Command\+::timeout\+\_\+seconds = default\+\_\+timeout}

How long to run until we cancel this command. If the command is cancelled, \doxylink{classAutoCommand_a68475854d11e7c3be90e9c6e8fc9e982}{on\+\_\+timeout()} is called to allow any cleanup from the function. If the timeout\+\_\+seconds \texorpdfstring{$<$}{<}= 0, no timeout will be applied and this command will run forever A timeout can come in handy for some commands that can not reach the end due to some physical limitation such as
\begin{DoxyItemize}
\item a drive command hitting a wall and not being able to reach its target
\item a command that waits until something is up to speed that never gets up to speed because of battery voltage
\item something else... 
\end{DoxyItemize}
