\doxysubsection{Detailed Description}
\doxylink{classOdometryTank}{Odometry\+Tank} defines an odometry system for a tank drivetrain This requires encoders in the same orientation as the drive wheels Odometry is a "{}start and forget"{} subsystem, which means once it\textquotesingle{}s created and configured, it will constantly run in the background and track the robot\textquotesingle{}s X, Y and rotation coordinates.
